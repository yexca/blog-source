## ngrok OneClick

1. 到[官网](https://ngrok.com/)注册账号并验证邮箱后会获得 *Authtoken*

2. 下载程序：[ngrok - download](https://ngrok.com/download)

3. 添加 Token

```bash
ngrok config add-authtoken YourToken
```

4. 运行

```bash
ngrok http [Port]
```

替换 [Port] 为要监听的本地端口
